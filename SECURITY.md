# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within MOFU, please send an email to the maintainers. All security vulnerabilities will be promptly addressed.

## Security Best Practices

When using MOFU in production:

- Keep dependencies updated
- Use the latest stable version
- Review security advisories
- Follow Go security best practices

## Scope

This security policy applies to the MOFU framework itself, not to applications built with MOFU.
